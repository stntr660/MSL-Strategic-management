# Meeting Notes

Store all meeting minutes and notes in this folder.

## How to Use
1. Copy the [Meeting Notes Template](../templates/Meeting-Notes-Template.md) to this folder
2. Rename with date and meeting type
3. Fill out all sections during or after the meeting
4. Link to related strategic plans and action items

## Naming Convention
Use format: `YYYY-MM-DD-Meeting-Type.md`
Examples: 
- `2024-03-15-Board-Meeting.md`
- `2024-03-20-Strategy-Session.md`
- `2024-03-25-Team-Weekly.md`

## Organization Tips
- Tag meetings with relevant topics: #strategy #finance #hr
- Use internal links to connect related meetings
- Include action items with clear ownership and deadlines
