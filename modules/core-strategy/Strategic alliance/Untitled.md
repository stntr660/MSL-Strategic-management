## ✅ Updated Deliverables (medical equipment)

Each product entry must include:

- **Category / Subcategory** (standardized taxonomy)
    
- **Product Name (FR / EN)**
    
- **Reference / SKU**
    
- **Manufacturer**
    
- **Description (FR / EN)**
    
- **Brochure (PDF)** — **prefer French**; fallback to English. Store PDF file and source URL.
    
- **Primary Image**
    
- **Secondary Images**
    
- **Accuracy Score** (includes brochure presence & language preference)
    
- **Source URL**
    
- **Scrape Report Fields** (`scraped: true/false`, `completeness_score`, `missing_fields`, etc.)
