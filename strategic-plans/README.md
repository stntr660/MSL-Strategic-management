# Strategic Plans

This folder contains strategic planning documents for MSL.

## How to Use
1. Copy the [Strategic Plan Template](../templates/Strategic-Plan-Template.md) to this folder
2. Rename it with an appropriate title and date
3. Fill out all sections
4. Link to related meeting notes and project proposals

## Naming Convention
Use format: `YYYY-QX-Strategic-Plan-Title.md`
Example: `2024-Q2-Digital-Transformation-Plan.md`
